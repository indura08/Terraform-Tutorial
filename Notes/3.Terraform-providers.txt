âœ… Versioning Providers
-------------------------
You can specify versions to avoid breaking changes:

  required_providers {
    aws = {
      source  = "hashicorp/aws"
      version = "~> 5.0"
    }
  }
}

This locks the AWS provider to versions like 5.x.x.
